## Requirements:
### 1. Python
Python 3.11.5 or above.

### 2. PyQt6
Install using cmd or virtual environment console.
```ShellSession
pip install PyQt6
```

## How to Use:
### Finite Population.
- Enter population
- Enter confidence level
- Enter success probability

### Infinite Population.
- Enter confidence level
- Enter success probability
